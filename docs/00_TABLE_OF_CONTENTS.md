## How to Use This Tutorial

1. **Start from `01`** if you are brand new to this template.
2. **Jump to `07`** if you already understand the basics and want to add a new feature.
3. **Go to `08` or `09`** if you need API or Socket integration specifically.
4. **Use `10`** as a hands-on coding exercise to build a full feature end-to-end.
5. **Keep `11`** open as a cheat-sheet whenever you're looking at the code.

Happy coding! 🚀
